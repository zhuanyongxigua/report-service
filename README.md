## Getting Started

#### Data storage

* mongodb

#### Quick Start

1. Get the latest version

   ```
   $ git clone git@github.com:zhuanyongxigua/wechatForum.git
   $ cd wechatForum
   ```

2. Run npm install

   ```
   npm install
   ```

3. Run dev

   ```
   npm run dev
   ```

4. Run product

   ```
   npm start
   ```
